# אלעד יעקובוביץ - אתר תיק עבודות

אתר תיק עבודות ופורטפוליו מודרני של מפתח Full-Stack, בנוי עם Next.js 16. כולל בלוג עם MDX, תצוגת פרויקטים, רכיבי UI אינטראקטיביים, ותמיכה מלאה בעברית ואנגלית.

## תכונות

- תיק עבודות עם תצוגת פרויקטים
- בלוג מבוסס MDX
- תמיכה דו-לשונית (עברית RTL + אנגלית)
- עיצוב רספונסיבי Mobile-First
- מצב כהה/בהיר
- אנימציות Framer Motion
- בדיקות E2E עם Playwright

## טכנולוגיות

| טכנולוגיה | שימוש |
|-----------|-------|
| Next.js 16 | פריימוורק React |
| TypeScript | בטיחות טיפוסים |
| Tailwind CSS | עיצוב |
| MDX | תוכן בלוג |
| Radix UI | רכיבי UI נגישים |
| Framer Motion | אנימציות |
